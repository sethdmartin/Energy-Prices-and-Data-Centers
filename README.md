# Energy-Prices-and-Data-Centers
Comparing recent changes in energy prices to the emerging abundance of data centers in the US.

US_en_price_changes_plot_by_state.py: Generates a plot color-coding the United States according to residential energy price changes, and provides data center energy usage by state.

This project is in its early stages, we do not expect obvious trends between such complex variables without sufficiently complex analysis. The existing code is to provide background and allow exploration of the data in preparation of deeper analysis.
